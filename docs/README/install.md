# 安装

### NPM&YARN

```shell
npm install animate.css --save
```

```shell
yarn add animate.css
```

### CDN

```html
<head>
    <link rel="stylesheet"href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.0.0/animate.min.css"/>
</head>
```

### bounce

> [!TIP|style:flat|label:Code]
>
> <span class="tip">Old</span>
>

> <p class="animated infinite bounce">bounce</p>
> ```
> 
> <span class="tip">New</span>
>

> <p class="animate__animated infinite animate__bounce">bounce</p>
> ```
